# InkSight 功能清单 (Features)

InkSight 是一款集深度阅读、思维导图与知识管理于一体的现代化 Web 应用。它旨在帮助用户更高效地处理信息，将阅读转化为知识。

## 核心功能 (Core Features)

### 1. 多格式文档阅读 (Multi-format Reader)
- **PDF 阅读**: 基于 PDF.js 的高性能 PDF 渲染。
  - 支持缩放、跳转页面。
  - **文本选择**: 支持标准文本选择与复制。
  - **矩形选择 (Rectangle Select)**: 框选截图，用于捕捉图表或公式。
  - **椭圆选择 (Ellipse Select)**: 强调特定区域。
  - **荧光笔 (Highlighter)**: 自由绘制高亮，支持调节粗细。
- **EPUB 阅读**: 基于 epub.js 的电子书阅读体验。
  - 支持流式排版与章节跳转。
- **纯文本/Markdown 阅读**: 支持 `.txt` and `.md` 文件。
  - Markdown 实时渲染预览。

### 2. 智能思维导图 (Intelligent Mind Map)
- **卡片式节点**: 每个节点都是一张信息卡片。
- **拖拽创建**: 从阅读器中选中文本或截图，直接拖拽到思维导图区域生成节点。
- **双向链接 (Backlinks)**:
  - 点击思维导图节点，自动跳转回文档对应的原始位置（页码/段落）。
  - 即使文档未打开，也能显示节点来源的文档名称。
- **自动布局 (Auto Layout)**: 集成 ELK 布局算法，支持正交路由与节点自动避让，一键生成清晰的树状或网络结构。
- **Drawnix 集成**: 采用 Drawnix 绘图引擎，支持无限画布、自由绘图与节点连接。

### 3. 知识管理与持久化 (Knowledge Management)
- **本地优先**: 所有数据存储在本地（IndexedDB/LocalStorage），保护用户隐私。
- **自动保存**: 实时保存阅读进度、高亮标注与思维导图状态。
- **文件管理**: 侧边栏文件列表，支持多文件切换。
- **导入/导出**: 支持导出 `.drawnix` 格式的项目文件，包含所有高亮与导图数据。

### 4. 现代化 UI/UX
- **分屏设计**: 左侧阅读，右侧笔记，支持自由调节比例。
- **响应式布局**: 适配不同屏幕尺寸。
- **暗色模式/主题**: (计划中) 支持多种配色主题。
